# User Management Dashboard

A simple React.js User Management Dashboard where you can:
- Add new users
- View user details
- Delete and edit users
- Switch between light/dark mode
- Get toast notifications

## 🛠️ Technologies Used
- React.js
- Tailwind CSS
- react-router-dom
- react-hot-toast

npm install
npm run dev
